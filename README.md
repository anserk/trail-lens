# Trail Lens

Offline-first hiking companion for identifying plants, trees, animals, and eventually sounds/video using on-device machine learning.

## Vision

Trail Lens aims to:

* Take a photo of a plant/tree/animal
* Run ML inference locally on a mobile phone
* Receive likely species matches with confidence scores
* Save sightings for later review

Future goals include:

* Bird/animal sound classification
* Video support
* GPS-aware predictions
* Offline model inference
* Species comparison and identification help
